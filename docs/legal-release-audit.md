# Legal Release Audit

Audit date: 2026-06-30.

## Scope

This audit covers the public release boundary for the Foundry VTT v14 system package: repository-tracked runtime files, generated compendium packs, `module/data/complete-rules.json`, and the release ZIP generated by `npm.cmd run release`.

## Release Boundary

The release ZIP is intended to ship only runtime-safe Foundry system files and curated play aids. It must not include local PDFs, extraction caches, scratch conversion folders, raw source scans, or unedited source-book text.

Rules-reference compendia are limited to short lookup summaries, source page metadata, and workflow pointers. They are not a replacement for the original Part-Time Gods Second Edition rulebook.

## Automated Guard

`npm run validate` now fails when rules-reference pages lack `safeSummary` flags, contain repeated placeholder boilerplate, contain known PDF extractor artifacts, fall below the minimum useful-summary length, repeat the same rules paragraph, or grow beyond the release-safe summary threshold.
